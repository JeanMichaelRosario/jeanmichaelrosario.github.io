# Palabras de Canciones (sin internet)

Generador de palabras para juegos en familia, inspirado en vocabulario común en letras en español.

## Correr local
> El modo offline (Service Worker) funciona mejor en `http://localhost` o en hosting HTTPS (GitHub Pages).

### Python
```bash
python -m http.server 5173
```
Abre: http://localhost:5173/palabras-canciones/

### Node
```bash
npx serve .
```

## GitHub Pages
Si pones esta carpeta dentro de tu repo, asegúrate de entrar a la URL final una vez para que se cachee.

Ejemplo:
`https://TU_USUARIO.github.io/TU_REPO/palabras-canciones/`
